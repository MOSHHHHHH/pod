שלב 1 — התקנת "המנוע" ב-Apps Script

קובץ ה-GS הוא נקודת הכניסה שמתזמנת את ההורדות בכל לילה. הוא קצר בכוונה — כל הלוגיקה נמצאת בקובץ ה-JS שתעלה ב-Drive.

⚡ אפשרות 1 — קישור ישיר

פתח את הפרויקט המוכן ישירות בחשבון Google שלך ועבור לשלב ההרצה.

🔗 פתח Apps Script

🛠️ אפשרות 2 — ידני

הורד את קובץ הקוד, צור פרויקט חדש ב-Apps Script והדבק את התוכן.

🛠️ הנחיות הרצה וקבלת הרשאות:
1
בעורך Apps Script בחר את הפונקציה setUp בתפריט הנפתול ולחץ ▶ Run.
2
ייפתח חלון "Authorization required" — לחץ Review permissions ובחר את חשבון Google שלך.
3
אם מופיע "Google hasn't verified this app" — לחץ Advanced ואחר כך Go to ... (unsafe). זה בטוח — זה הפרויקט הפרטי שלך.
4
לחץ Allow לאישור כל ההרשאות. המערכת תיצור עכשיו את התיקיות ב-Drive.
💡 לאחר ההרצה הראשונה המערכת תיצור אוטומטית את התיקייה פודקאסטים 2.0 / קבצי מערכת ב-Drive שלך. שם תצטרך להעלות את שני הקבצים בשלב 3.
